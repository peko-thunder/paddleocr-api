## フォント詳細

### Noto Sans JP / Noto Serif JP
- **開発元**: Google
- **特徴**: Googleが開発した多言語対応フォント。日本語を含む多くの言語をカバー
- **用途**: Webフォント、UI、印刷物など幅広く使用
- **URL**: https://fonts.google.com/noto

### 源ノ角ゴシック (Source Han Sans)
- **開発元**: Adobe, Google
- **特徴**: AdobeとGoogleが共同開発。Noto Sans CJKと同一のグリフ
- **用途**: デザイン、DTP、Web
- **URL**: https://github.com/adobe-fonts/source-han-sans

### IPAexフォント
- **開発元**: 情報処理推進機構 (IPA)
- **特徴**: 日本政府機関が開発した高品質な日本語フォント
- **用途**: 公文書、学術文書、一般文書
- **URL**: https://moji.or.jp/ipafont/

### M+ FONTS
- **開発元**: M+ FONTS PROJECT
- **特徴**: 日本語と基本ラテン文字をカバーするモダンなデザイン
- **バリエーション**:
  - 1p: プロポーショナル（可変幅）
  - 1c: 固定幅
  - 2p: より丸みのあるデザイン
- **URL**: https://mplusfonts.github.io/
